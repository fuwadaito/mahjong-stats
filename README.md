# JAN STA (mahjong-stats)

A simple and intuitive web application to record and analyze four-player mahjong match metrics. 
麻雀（四人麻雀）の対局結果を記録し、スタッツを可視化・分析するためのWebアプリケーションです。

---

## 🚀 Features (主な機能)

* **Match Result Recording**: Input placement and scores seamlessly. (対局順位や素点の入力・記録)
* **Performance Metrics**: View key statistics such as average placement and top-two finish rates. (平均順位や連対率などの主要スタッツ自動計算)


## 🛠️ Tech Stack (技術スタック)

* **Frontend**: HTML5 / CSS3 / JavaScript (Vanilla JS)
* **Style Framework**: `sanitize.css` (for cross-browser consistency)
* **Hosting**: Netlify

## 📂 File Structure (ファイル構成)

* `index.html` - The main structure of the application.
* `app.js` - Core logic for match recording and statistics tracking.
* `app-newmatch.js` - Logic handling the input and creation of new match records.
* `style.css` / `sanitize.css` - Custom styling and layout adjustments.
